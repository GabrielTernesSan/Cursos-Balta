## MIN, MAX, Count

### MIN

````sql
SELECT TOP 100
	-- Para pegar o minimo do campo
    MIN([Id])
FROM    
    [Curso]
````

### MAX

`````sql
SELECT TOP 100
	-- Para pegar o maximo do campo
    MAX([Id])
FROM    
    [Curso]
`````

### COUNT

````sql
SELECT TOP 100
   COUNT([Id])
FROM    
    [Curso]
````

### MEDIA (AVERAGE)

````sql
SELECT TOP 100
   AVG([Id])
FROM    
    [Curso]
````

### SOMA (SUM)

```sql
SELECT TOP 100
   SUM([Id])
FROM    
    [Curso]
```
